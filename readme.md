This project heavily utilizes the Komodo package provided by NASA.

Komodo utilizes Python 3.11 and various outdated versions of popular packages.

It is heavily recommended to follow the instructions provided by NASA to initialize your own conda environment with a Komodo installation inside it.

Instructions are found here:

[https://nasa.github.io/Kamodo/notebooks/QuickStart.html](https://nasa.github.io/Kamodo/notebooks/QuickStart.html)
